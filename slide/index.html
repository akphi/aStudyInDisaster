The Diagnostics

Hosmer-Lemeshow goodness-of-fit test

  • Treating [0,1] as probability range
  • Divide into bin(s)
  • Conduct goodness-of-fit test on each bin
  • The null hypothesis is: “The observed and the expected proportions are the same for each bin”
